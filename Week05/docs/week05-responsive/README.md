# Week05 Responsive Screenshots

Thư mục này chứa screenshot minh chứng layout responsive tại 3 breakpoints.

## Breakpoints cần chụp

| Breakpoint | Độ rộng | Grid phòng | Navbar |
|-----------|---------|-----------|--------|
| Mobile    | 375px   | 2 cột (`xs={2}`) | Hamburger menu (collapsed) |
| Tablet    | 768px   | 3 cột (`sm={3}`) | Hamburger menu (collapsed) |
| Desktop   | 1280px  | 4–5 cột (`md={4}`, `lg={5}`) | Navbar full hiển thị |

## Cách chụp thêm screenshot trong Chrome DevTools

1. Mở trình duyệt tại `http://localhost:5174`
2. Nhấn `F12` → chọn tab **Device Toolbar** (icon điện thoại)
3. Nhập độ rộng: **375**, **768**, **1280**
4. Nhấn `Ctrl+Shift+P` → gõ `Capture screenshot` → lưu vào thư mục này
